## Asteroids ##

Simple spaceship in asteroid field game.

Developed with my custom C++ engine.

* **OpenGL3**
* GLFW
* GLAD
* **GLM**
* Custom **physics and collisions**
* Generic entity **pool**
* Custom **vertex** shader, **geometry** shader and **fragment** shader.
* FreeType
* VS2017
